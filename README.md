## 🎯 What is Data Entry Bot?

**Data Entry Bot** is a powerful yet simple command-line application that helps you manage, store, and organize data efficiently. Whether you're managing customer lists, employee records, or any structured data, this bot makes data entry fast and error-free.

## ✨ Why Use This Bot?

Tired of manual data entry in spreadsheets? This bot automates the process with:

- **⚡ Speed** - Add records in seconds
- **✅ Accuracy** - Automatic email and phone validation
- **💾 Persistence** - Data saved automatically to JSON
- **🔄 Flexibility** - Import/Export CSV files
- **🔍 Search** - Find records instantly

## 🚀 Quick Start

### Prerequisites
- Python 3.6 or higher
- pip package manager

### One-Click Setup


# Clone the repository
git clone https://github.com/ElishaKongolo/data-entry-bot.git

# Navigate to project
cd data-entry-bot

# Install dependencies
pip install pandas numpy

# Run the bot
python data_entry_bot.py
